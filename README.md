# CletaEats — Web Frontend

Panel de administración web para CletaEats. Desarrollado en React con la misma paleta de colores y funcionalidades de la app móvil.

## Tecnologías
- React 18
- React Router DOM v6
- react-hot-toast (notificaciones)
- lucide-react (íconos)
- CSS Variables (sistema de diseño)

## Requisitos
- Node.js 16+
- npm 8+

## Instalación y uso

```bash
npm install
npm start
```

La app corre en http://localhost:3000

## Build para producción

```bash
npm run build
```

## Funcionalidades

| Sección | Funcionalidades |
|---------|-----------------|
| Login | Autenticación con usuario y contraseña |
| Dashboard | Estadísticas generales, hora pico, cliente estrella |
| Clientes | Listar activos/suspendidos, registrar, activar/suspender |
| Repartidores | Listar, registrar, ver quejas, agregar amonestaciones |
| Restaurantes | Listar, registrar, ver menú con mapa integrado |
| Pedidos | Flujo completo: crear, agregar combos, confirmar, facturar, cambiar estado |
| Reportes | 6 reportes: mayor/menor pedidos, montos, cliente top, hora pico |
